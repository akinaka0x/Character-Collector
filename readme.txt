Character-Collector
手書き文字データを収集するためのアプリです。
Copyright (C) 2017 中村晶洋

個人で自由に使ってよいですが、再頒布はしないでください。

使い方
	Character-Collector.exeを起動して、説明通りに文字を書いてください。
	もしセキュリティソフトなどが警告を出しても無視してください。
	文字が赤線にできるだけかぶらないようにお願いします！
	終了後に
		image_kakezan_(数字)_(数字).png
		image_warizan_(数字)_(数字).png
	が生成されるので、Google ドライブの”出力画像”フォルダに
	この2つのファイルをアップロードしてください。
	手書き文字データ収集のため、ご協力お願いします。
	
使用ライブラリ
	OpenCV-2.4.13
	Visual Studio 12.0  Visual C++ ランタイムライブラリ
